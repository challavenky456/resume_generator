# Simple Resume Generator

## 📌 Project Description
The Simple Resume Generator is a web application that allows users to create a basic resume by entering their personal details. The application collects information such as name, email, skills and education through a form and automatically generates a formatted resume page.
This project demonstrates the integration of frontend and backend technologies using HTML, CSS and Python Flask.

## 🚀 Features
- User-friendly resume input form
- Automatic resume generation
- Simple and clean interface
- Beginner-friendly web development project

## 🛠 Technologies Used
- HTML – Structure of the webpage
- CSS – Styling and layout
- Python – Backend programming
- Flask – Python web framework

## 📖 How It Works
1. The user opens the web page.
2. The user enters personal details in the form.
3. When the user clicks **Generate Resume**, the data is sent to the Flask backend.
4. The application processes the data and displays the generated resume.
